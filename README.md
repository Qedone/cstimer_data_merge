# cstimer_data_merge
一个自动合并 csTimer 备份文件的 Python 轻量脚本

当你在不同设备上使用 csTimer 记录了不同的新成绩，这个工具可以帮你将两个备份文件中的成绩完美合并——自动补充缺失的数据，且不会产生重复记录。

## 核心功能

* **自动识别文件**：将脚本和两个需要合并的 `.txt` 备份文件放在同一目录下即可识别。
* **智能分组匹配**：跨文件寻找同名分组（忽略大小写和前后空格）。即使在两个文件中分组的内部 ID 不同，只要名字一样就能准确匹配。
* **严格去重防乱**：采用 `(时间戳 + 打乱公式)` 作为唯一标识，确保原有成绩绝对不会重复，并自动按时间顺序重新排列。
* **缓存自动修复**：合并后自动清理 csTimer 的旧统计缓存，确保导入后左侧边栏的成绩总数和统计数据计算准确无误。

## 使用方法

### 1. 准备环境
确保你的电脑上已安装 [Python 3](https://www.python.org/downloads/)。

### 2. 准备文件
新建一个空文件夹，将以下文件放入该文件夹中：
1. 本脚本文件 `merge.py`
2. 你的两份 csTimer 备份文件（即在不同设备上导出的`.txt`文件）

> **注意**：为了防止合并错误，请确保该文件夹下**只有 2 个**备份的 `.txt` 文件。如果有更多文件，脚本会主动报错并停止运行。

### 3. 运行脚本
打开终端（或命令行），进入该文件夹，运行以下命令：

```bash
python3 merge.py
