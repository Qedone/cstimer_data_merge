# CubeInsight

CubeInsight 是一款专为魔方爱好者打造的 **csTimer 成绩可视化与数据处理工具**。
无需安装任何软件，支持一键生成专业的成绩趋势图表，并提供多设备导出的成绩文件智能合并与区间过滤功能。

## 核心特性

### 1. 成绩趋势总结
* 自动读取 csTimer 导出文件（`.txt`），还原你设置的真实分组名称。
* 高透明度散点精准还原成绩分布，成绩越密集的区间颜色越深。
* 自动剔除 DNF 成绩，精准计算 `+2` 罚秒，展现每日纯净的算术平均进步曲线。
* 鼠标悬浮至某日区域，上方信息栏会自动计算并显示该日的**最佳 Single**、**最佳 mo3**、**最佳 ao5** 以及当日练习次数。

### 2. 数据处理与合并
* 支持自定义开始与结束日期，一键剔除指定区间外的冗余数据，导出纯净成绩。
* 支持上传两个 csTimer 备份文件，底层采用 `时间戳 (timestamp) + 打乱公式 (scramble)` 作为唯一标识，完美解决多设备游玩后的成绩合并去重问题。
* 执行处理后，会自动生成直观的数据表格，详细展示各分组的**新增条数**、**重复跳过条数**、**过滤条数**以及**最终条数**。
* 处理完成自动触发下载，生成标准格式的 `merged_YYYYMMDD_HHMM.txt` 文件，可直接无缝导入回 csTimer。

## 隐私与安全
CubeInsight 是一个 100% 的纯前端 Web 应用（Vanilla JS + HTML + CSS）。所有的数据读取、图表渲染、文件合并与去重均在本地浏览器中完成。没有任何数据会被上传到云端服务器，请放心使用。

## 技术栈

* **核心**：HTML5 / CSS3 / JavaScript (ES6+)
* **图表库**：[Apache ECharts 5.5.0](https://echarts.apache.org/) (CDN 引入)
* **设计风格**：马卡龙薄荷绿 (Mint Green) 卡片式极简设计，全响应式适配移动端与桌面端。

## 详细操作指南

**查看成绩图表：**
1. 在网页中选择“成绩趋势”标签。
2. 上传你从 csTimer 导出的 `.txt` 备份文件。
3. 在上方下拉框选择你想查看的魔方分组。
4. 使用底部的滑块或日期输入框，随意缩放你想查看的日期范围。

**合并/过滤成绩：**
1. 切换至“数据处理”标签。
2. **截取模式**：仅上传“基础文件”，设置日期区间，点击执行即可过滤。
3. **合并模式**：同时上传“基础文件”与“补充文件”，不填日期即为全量去重合并；填写日期则为合并后再按日期过滤。
4. 查看下方的预览统计表确认无误后，使用浏览器自动下载的新文件导入 csTimer 即可。
